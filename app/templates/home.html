Measure four systems. Build one behaviour profile.

BodyMap Net lets you run any combination of four independent fitness tests, then fuses whatever you completed into a single overall Behaviour Profile using a neural network built to tolerate missing data — and conditioned on your stated goal, so the same body can read differently depending on what you're training for.

How this works

Each week, you submit any combination of the four tests below. Every submission is saved against that calendar week, so BodyMap Net builds up a week-by-week history rather than a single snapshot.

  • Predict — each test you run goes through its own small neural network to produce a band/label (e.g. "Balanced", "Normal", "On-target"), and if you tick the fifth option, a fusion model combines whichever tests you ran into one overall Behaviour Profile.
  • Judge against your goal — a shared rulebook decides whether each result is actually good news for your stated goal (a calorie deficit is a win if you're trying to lose fat, and a problem if you're trying to gain muscle) — this same rulebook drives the goal-verdict banner, the tips, and the streaks, so they never contradict each other.
  • Track over time — streaks count consecutive weeks a metric stayed good or bad, and your full history is always one click away.

See Test > Model Components for what each model actually looks at, and History > How it saves for how the weekly storage and streaks work.

The four tests

01

Exercise Test

Enter this week's training volume across up to 44 exercises (all optional). Returns a training-adequacy read: under-training, balanced, or over-training.

02

BMI Test

Enter height and weight to place you across the standard BMI bands.

03

Body Fat Test

Enter neck / waist (and hip, if applicable) measurements for a body-fat band estimate.

04

Ideal Calorie Test

Enter activity level and current intake to compare against your Mifflin-St Jeor target.

Behaviour Profile (AI)

Tick the fifth option and a fusion neural network combines whichever of the four tests you completed — one, two, three, or all four — into a single behaviour-profile prediction: Obesity Risk, Buff / Muscular, Skinny Athlete, Average / Balanced, or Endurance-Focused. It's conditioned on your stated goal, so it isn't just "what body type is this" — it's "does this match what you're training for". Because it uses learned "missing-branch" signals rather than zeros, it also degrades gracefully when tests are skipped.

This is a demonstration project, not a medical or security-grade system. Accounts are created by an admin and protected by a password; use the "Log in" button in the top corner to sign in as a user or as an admin.
